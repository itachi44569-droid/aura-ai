How customers use their key

Tell your customer to include their key in the chat request. If they use the website, they paste it in the "API Key" field.
If they call the API directly: 

POST /chat
{
  "user_id": "their_id",
  "message": "Hello!",
  "api_key": "nb-xxxxxxxxxxxx"
}

Free users (no key) are limited to 20 messages/day. Paid keys bypass this.
