FAQ

Common questions.

Do I need to pay for any APIs to run this?+
No. Every API used is completely free. Groq (AI), Pollinations (images), DuckDuckGo (search), Open-Meteo (weather) — all free with no credit card required. Railway's free tier is enough to host the whole app.
Can I use my own domain name?+
Yes. Railway supports custom domains. After purchasing, you can point any domain you own to your Railway deployment. The deploy guide includes step-by-step instructions.
Can I resell or charge users for access?+
Yes. After purchase, this code is yours. You can rebrand it, charge users for access, sell it as a SaaS, or use it as a client project. The only thing you can't do is resell the source code itself as a template.
What do I need to know to deploy this?+
Basic comfort with the terminal and a Railway account (free). The deploy guide walks you through everything — no prior Python or FastAPI experience needed. If you want it set up for you, get the Template + Setup tier.
What AI model does it use?+
Llama 3.3 70B via Groq — one of the best open-source language models available. It's free on Groq's free tier. You can also configure it to use any other LLM by changing a few lines in the backend.
